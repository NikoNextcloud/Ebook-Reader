# Voxora AI Reader

Responsive React PWA за превръщане на потребителски текст в аудио преживяване.

## Стартиране

1. Стартирайте `START-VOXORA.bat`.
2. Отворете адреса, показан в прозореца (обикновено `http://localhost:4173`).

Приложението използва наличните гласове на устройството. Българските гласове зависят от операционната система и браузъра. PDF, DOCX и TXT се обработват локално и не се изпращат към сървър.

За реален cloud AI TTS заменете browser provider-а в `src/services/speechService.js`, без промяна в интерфейса.
